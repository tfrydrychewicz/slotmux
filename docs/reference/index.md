# Reference

## API (generated)

The full TypeScript API for the `slotmux` package is generated into **`reference/api/`** when you run `pnpm docs:api` or `pnpm docs:build` from the repository root.

- [API index](/reference/api/README)

## Snippets

Runnable documentation snippets live under `docs/snippets/` and are compiled with `pnpm test:docs` in CI.
